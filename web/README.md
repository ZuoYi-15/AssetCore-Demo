# Asset-Core Web

Vue frontend for Asset-Core trusted device asset management.

## Stack

- Vue 3
- Vite
- TypeScript
- Element Plus
- lucide-vue-next
- Axios

## Start

```powershell
npm install
npm run dev
```

The dev server runs at:

```text
http://127.0.0.1:5173
```

Vite proxies `/api` and `/health` to `http://127.0.0.1:8080`.
